Cloud Research

A member of a team researching and implementing Amazon AWS cloud services.   Our goal is to make a multiplayer application that utilizes cloud storage and messaging services.   So far, we have developed the logic behind the game and have a working command line version.   We also have a MySQL database created which will store info for current games and active users in the system.   Our next step is to integrate the game with the database and research threading, which will be used to regulate how often the program checks the database.
